# Uuid

Генерирует случайный uuid.

WARNING

Криптографисеская стойкость отсутствует

# Базовое использование

import {generators} from 'jseeder';

generators.uuid().get() // '03e6cde0-5dea-4579-a9ba-802a685f9654'
